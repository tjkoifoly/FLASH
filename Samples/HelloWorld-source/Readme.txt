Read the following before using the file within this archive.

This archive contains a file that belongs to the following article on the Adobe Developer Connection:
http://www.adobe.com/devnet/flash-builder/articles/hello-world.html

Unpack the archive and use the files as follows:

* HelloWorld.fxp: Import this file into Flash Builder (select New > Import Flash Builder Project) so you won't have to enter or copy the code shown in the article. This version shows the default button styles, as described through the end of the section, "Testing the app on the desktop." 

* HelloWorldSkins.fxp: Import this file according to the directions in the section, "Bonus: Redesigning the buttons with skins," to follow along with the last section of the article.

(These files were updated for the Adobe Flash Builder 4.6 release.)
